Zastosowanie aplikacji

Aplikacja służy do pobierania, analizy i wizualizacji opinii na temat produktów na podstawie ich ID w serwisie ceneo.pl

Opis działania aplikacji

Użytkownik wprowadza kod produktu z Ceneo.pl, na podstawie którego aplikacja automatycznie pobiera wszystkie dostępne opinie wraz z szczegółowymi informacjami (autor, ocena gwiazdkowa, rekomendacja, treść, wady, zalety, daty itp.). Zebrane dane są następnie prezentowane w postaci interaktywnej tabeli z możliwością sortowania i filtrowania oraz eksportowane do różnych formatów plików

Zastosowane biblioteki

  • Flask - framework do tworzenia aplikacji webowych
  • DataTables - biblioteka JavaScript do zaawansowanych tabel interaktywnych
  • jQuery - biblioteka JavaScript do manipulacji DOM
  • Bootstrap - biblioteka JavaScript do dynamicznych treści, CSS
  • BeautifulSoup/requests - biblioteki Python do web scrapingu (pobierania danych z Ceneo.pl)
  • pandas/csv/json - biblioteki do przetwarzania i eksportu danych

Zastosowane podejścia

  • Strukturalne
  • Obiektowe
